# Cycle 8 — Learning Report

## What Worked
1. **Emotional features are cheap but impactful**: 20 template strings + name input = personalized daily experience
2. **Placeholder system is extensible**: Can add {dday}, {streak}, {tasks_done} to messages later

## Key Insight
The product now has both hemispheres:
- **Left brain**: Tasks, budget, timeline, urgency (practical)
- **Right brain**: Journey, achievements, couple names, warm messages (emotional)

A complete wedding OS needs both. Most competing apps only have the left brain.
