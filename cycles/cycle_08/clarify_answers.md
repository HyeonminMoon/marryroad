# Cycle 8 — Clarify Answers

1. **Emotional gap**: The app tracks what you've DONE but doesn't celebrate WHO you're doing it for. Wedding prep is stressful — the app should occasionally remind the user WHY they're doing all this.

2. **Journey is passive**: It shows history but doesn't prompt reflection. A "couple story" or memory prompt could make it active.

3. **Wow moment**: A "Love Letter" feature — the app periodically surfaces a warm message or prompts the user to write one for their partner. Simple but deeply emotional.

4. **Untapped**: The transition between "task completed" and "next task" is purely mechanical. What if completing certain milestone tasks triggered a personalized celebration — not just confetti, but a meaningful message?

5. **Smile trigger**: A daily motivational message on the roadmap page. Something like "오늘도 [파트너이름]과의 미래를 위해 한 걸음 더 👣" — personalized, warm, rotating.

## Decision
**Couple Profile + Daily Motivation Message**

MVP:
- Set partner names (me + partner)
- Daily rotating warm message using names
- Milestone celebrations with personalized text
- Small but emotionally powerful

This is low effort but creates a personal connection every time the user opens the app.
