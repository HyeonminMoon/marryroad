# Cycle 69 — Design Spec

## 수정 사항

### 1. 헤더 드롭다운
Before: 프로필(/profile, 깨진) + 설정(/settings)
After: 설정(/settings) 하나만

### 2. 하객 Empty State
Before: "하객 명단을 추가해보세요"
After: "하객 명부를 만들어보세요" + "참석 인원, 식사 선택, 축의금까지 한눈에 관리할 수 있어요"

### 3. 캘린더 Empty State
Before: "로드맵에서 작업을 시작해보세요!"
After: 날짜 설정 여부에 따라:
- 있음: "아직 일정이 비어있어요" + "태스크 완료하면 타임라인이 자동으로 채워져요"
- 없음: "결혼 날짜를 먼저 설정해보세요" + "D-day가 정해지면 준비 일정이 달력에 자동 표시"

### 4. 데이터베이스 Empty State
Before: "검색 결과가 없습니다."
After: "조건에 맞는 태스크가 없어요. 필터를 조정해보세요."
