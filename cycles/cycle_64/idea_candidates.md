# Cycle 64 — Idea Candidates
## 필수 기능 감사 결과
UX Research CRITICAL/HIGH/MEDIUM 이슈 모두 완료 후, 핵심 기능 부재 점검.

1. **하객 관리 (Guest Management)** — 결혼식 앱의 핵심. 하객 목록, RSVP, 식사 선택, 축의금 추적
2. **설정 페이지** — 커플 이름, 결혼일, 예산 등 한곳에서 관리
3. **카테고리별 예산 배분** — 예산을 카테고리로 쪼개서 관리
4. **통계 대시보드** — 전체 진행률, 비용 분석 등 한눈에

## 선정 기준
- 결혼 앱으로서 "없으면 안 되는" 기능 우선
- 하객 관리가 1순위 — 모든 결혼 앱의 필수 기능
