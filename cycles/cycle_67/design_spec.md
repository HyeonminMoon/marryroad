# Cycle 67 — Design Spec

## SmartHomeSummary 컴포넌트

### 레이아웃 (3단 구조)
```
┌─────────────────────────────────────────┐
│ 🎯 결혼 준비 21% 완료                    │  ← 결과 프레이밍
│ "이 속도면 여유있게 준비할 수 있어요"      │  ← 페이스 피드백
│ ███████░░░░░░░░░░░░░░░░░░ 21%           │  ← 진행 바
├─────────────────────────────────────────┤
│ 💡 오늘의 추천                            │  ← 단일 추천 (이해 > 마찰)
│ "예식장 투어 일정 잡기" (웨딩홀 선정)      │
│ → 왜: 인기 웨딩홀은 1년 전 예약 필수     │  ← 맥락/이해 제공
│                              [시작] 버튼  │
├─────────────────────────────────────────┤
│ ✨ 최근 성과                              │  ← Aha moment 강화
│ 어제: "예산 계획 수립" 완료               │
│ 2일 전: "상견례 장소 리서치" 완료         │
└─────────────────────────────────────────┘
```

### 상태별 메시지

**첫 방문 (0% 완료)**
- "결혼 준비, 체크리스트가 아니라 여정이에요"
- "첫 태스크를 완료하면 여정이 시작됩니다"
- CTA: "첫 태스크 시작하기"

**초기 (1~20%)**
- "좋은 시작이에요! 기초가 탄탄하면 나중이 편해요"

**중반 (21~60%)**
- "절반을 향해 달려가고 있어요!"

**후반 (61~90%)**
- "거의 다 왔어요! 마무리만 남았습니다"

**완료 임박 (91~100%)**
- "완벽한 결혼식 준비가 거의 끝났어요!"

### 페이스 피드백 (D-day 기준)
- 여유: "이 속도면 여유있게 준비할 수 있어요 😊"
- 보통: "적절한 속도로 진행 중이에요 👍"
- 서두르기: "조금 서둘러야 할 수 있어요. 오늘 하나만 더! 💪"
- D-day 미설정: 페이스 피드백 생략

### 스타일
- Glassmorphism 카드
- 진행 바: purple gradient
- 추천 섹션: 파란색 좌측 보더
- 최근 성과: 연한 녹색 배경
