# Cycle 67 — Selected Feature
## Smart Home Summary (Lenny 인사이트 종합 적용)

매일 로드맵 페이지를 열면 가장 먼저 보이는 개인화된 요약 카드.

### Lenny 원칙 적용
- **결과 프레이밍**: "14개 중 3개 완료" → "결혼 준비 21% 완료! 이 속도면 여유있게 준비할 수 있어요"
- **단일 추천 액션**: "오늘 이것만 하세요" — 결정 피로 제거, 다음 행동 명확화
- **이해 > 마찰**: 현재 어떤 단계인지, 왜 이 태스크가 중요한지 맥락 제공
- **최근 성과 축하**: 어제/오늘 완료한 것 보여주기 (Aha moment 강화)
- **진행 맥락**: D-day 기준 페이스 피드백 ("여유 있어요" / "조금 서둘러야 해요")

### 범위
- SmartHomeSummary 컴포넌트 신규 생성
- 로드맵 페이지 최상단(Today 섹션 위)에 배치
- 진행률 기반 맥락 메시지 + 추천 태스크 + 최근 성과
