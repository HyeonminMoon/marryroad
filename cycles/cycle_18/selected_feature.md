# Cycle 18 — Selected Feature

## Quick Setup (캐치업 온보딩)

결혼 준비가 이미 진행 중인 유저를 위한 빠른 초기 설정.

### User Flow
```
/welcome Step 1: 커플 이름 입력
/welcome Step 2: 결혼 날짜 설정
/welcome Step 3 (NEW): "이미 진행 중인 항목이 있나요?"
  → "아니요, 처음 시작해요" → /roadmap
  → "네, 이미 준비 중이에요" → Quick Setup 화면
    → 19개 퀘스트 카드 목록
    → 각 퀘스트: [안 했어요] [진행 중] [다 했어요]
    → "다 했어요" → 해당 퀘스트 전체 일괄 완료
    → "진행 중" → 세부 태스크 토글 표시 (선택적)
    → 하단: "설정 완료" 버튼 → /roadmap
```

### Key Features
1. **퀘스트 레벨 빠른 마킹**: 19개만 보면 됨 (137개 아님)
2. **일괄 완료**: `bulkCompleteQuest` 액션 — 퀘스트 내 모든 태스크를 today 날짜로 완료
3. **부분 완료**: "진행 중" 선택 시 태스크 리스트를 펼쳐서 개별 체크 가능
4. **스킵 가능**: "처음 시작해요" 선택하면 기존과 동일한 플로우

### Why This Matters
- 한국 2030대 커플은 보통 결혼 준비를 6-12개월 전부터 시작
- 이 앱을 D-180에 발견한 사람은 이미 50%+ 완료 상태
- 앱을 열자마자 "나한테 맞는 도구"라는 느낌을 줘야 retention 확보
