# Cycle 18 — Clarify Questions

1. 결혼 준비가 이미 진행 중인 유저가 앱을 처음 쓸 때 어떤 경험을 하게 되나?
   - 137개 태스크가 전부 미완료 → "이거 다 해야 해?" → 이탈
   - 실제로 이미 예식장 계약, 스드메, 상견례 등 끝난 상태일 수 있음

2. 이미 진행된 태스크를 빠르게 "했어요" 처리할 수 있는 방법이 없음
   - 현재: 퀘스트 모달 열기 → 태스크 하나씩 체크 → 반복
   - 137개 중 50-60개 이미 한 사람은 이 과정만으로 이탈

3. 결혼 준비 "어디까지 했는지" 빠르게 셋업할 수 있는 온보딩이 필요한가?

4. 퀘스트 단위로 "이 퀘스트 전체 완료" 처리가 가능한가?

5. 웰컴 플로우에 "이미 준비 중이에요" 분기가 필요한가?
