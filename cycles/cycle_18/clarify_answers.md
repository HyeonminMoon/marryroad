# Cycle 18 — Clarify Answers

1. **현재 UX 문제**: 앱을 처음 연 사람은 무조건 "처음부터 시작하는 사람"으로 취급됨. 이미 예식장 잡고, 스드메 계약 끝내고, 청첩장도 보낸 D-90 유저가 이 앱을 발견하면? 137개 태스크 중 60개를 하나하나 체크해야 하는데, 이건 "도움이 되는 도구"가 아니라 "귀찮은 숙제".

2. **핵심 니즈**: "나 이미 여기까지 했어" → 빠르게 표시 → 남은 것만 보여줘. 이 과정이 1-2분 안에 끝나야 함.

3. **해결책: Quick Setup (캐치업 온보딩)**
   - 웰컴 플로우에서 "처음 시작" vs "이미 준비 중" 분기
   - "이미 준비 중" 선택 시 → 퀘스트 단위 체크리스트 표시
   - 각 퀘스트를 "안 했어요 / 진행 중 / 다 했어요"로 빠르게 마킹
   - "다 했어요" = 퀘스트 내 모든 태스크 일괄 완료 처리
   - "진행 중" = 퀘스트 모달로 이동해서 세부 선택 가능

4. **퀘스트 단위 일괄 완료**: quest-store에 `bulkCompleteQuest(questId)` 액션 추가. 퀘스트 내 모든 태스크를 한 번에 완료 처리하고, XP 및 activeDates 갱신.

5. **웰컴 플로우 확장**: 현재 2단계 (이름 → 날짜). 3단계로: 이름 → 날짜 → "어디까지 했나요?" (선택적). 이 3단계가 핵심.
