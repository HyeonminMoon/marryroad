# Cycle 19 — Selected Feature

## Task Detail Bottom Sheet (태스크 디테일 바텀 시트)

**Score: 10.5** (최고점)

### Why
- 사용자 직접 피드백: "디테일 내용이 ui상으로 좀 애매하다"
- 모바일 퍼스트 UX: 결혼 준비는 대부분 모바일에서 이루어짐
- 3중 중첩 해소: 모달 안의 인라인 폼 → 독립적 바텀 시트로 분리
- 시각적 계층 확보: 바텀 시트 내에서 섹션별 카드 구분

### What
1. 태스크 클릭 시 바텀 시트(Sheet) 오픈
2. 상단: 태스크 제목 + 상태 + 체크리스트
3. 중단: 카드형 섹션 (비용, 메모, 업체, 평점)
4. 하단: 완료/수정 버튼
5. 완료된 태스크는 읽기 모드로 표시, "수정" 버튼으로 편집 전환
6. 데스크톱에서는 사이드 시트(오른쪽 패널)로 동작

### Trade-offs
- (+) 정보 구조가 명확해짐
- (+) 모바일에서 자연스러운 인터랙션
- (-) 모달 안에서 시트를 또 열면 레이어가 깊어질 수 있음 → 시트를 모달 대체로 사용하는 방안 검토
- (-) Sheet 컴포넌트 필요 (shadcn/ui sheet 활용)
