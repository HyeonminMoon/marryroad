# MarryRoad UX Research — 10 Personas × Feedback

## Research Design
- 방법: Task-based Usability Testing + Think Aloud Protocol
- 기간: 가상 2주 사용
- 기기: 모바일 8명 / 데스크톱 2명
- 평가: Nielsen's 10 Heuristics + SUS (System Usability Scale)

---

## Persona 1: 김소연 (29세, 기획자)
**특성**: 꼼꼼한 계획형. 스프레드시트 마니아. 모든 걸 추적하고 싶음.
**기기**: iPhone 15 Pro

### 사용 패턴
- 매일 아침 "오늘" 탭 확인
- 모든 태스크에 비용+메모+공급업체 입력
- 데이터베이스 탭 자주 사용 (정렬/필터)
- 주 1회 Stats 탭에서 예산 분석

### 피드백
1. **[심각] 데이터베이스 테이블이 모바일에서 가로 스크롤 필요** — "6개 열이 화면에 안 들어가요. 작업명이 잘려서 뭔지 모르겠어요."
2. **[중간] 태스크 완료 시 비용 입력을 건너뛰면 나중에 찾기 어려움** — "Quick Complete 했다가 비용 입력 안 한 것들을 모아볼 수 없어요."
3. **[경미] Stats 탭에 컴포넌트가 너무 많음** — "WeeklyTrend → CompletionForecast → QuestComparison → Heatmap... 스크롤이 끝이 없네요."
4. **[개선] 데이터 Export가 JSON인데 Excel이면 좋겠음** — "남편이랑 공유하려면 스프레드시트가 편한데..."

## Persona 2: 박준혁 (32세, 개발자)
**특성**: 최소한의 입력. 게임화에 반응. 업적 수집가.
**기기**: Galaxy S24

### 사용 패턴
- Quick Complete 위주 (메모/비용 거의 안 남김)
- 퀘스트 맵에서 돌아다니며 클릭
- 업적 토스트에 관심 (XP 올리기)
- 주간 챌린지 적극 참여

### 피드백
1. **[심각] 주간 챌린지 "메모 남기기"가 억지로 느껴짐** — "메모 쓸 게 없는데 챌린지 때문에 아무 말이나 적게 돼요. 의미 없는 데이터."
2. **[중간] 퀘스트 맵(React Flow)이 모바일에서 너무 작음** — "노드가 작아서 정확히 탭하기 어려워요. 경로 모드는 괜찮은데."
3. **[중간] 업적 해금 시 어떤 조건으로 얻었는지 모름** — "갑자기 '첫 걸음' 뱃지가 뜨는데, 왜 얻었는지 설명이 없어요."
4. **[경미] 레벨업할 때 뭐가 달라지는지 모르겠음** — "Lv.3이 됐는데... 그래서? 혜택이 있나?"

## Persona 3: 이지은 (27세, 디자이너)
**특성**: 감성적. 예쁜 UI에 반응. 사진과 메모로 추억 기록.
**기기**: iPhone 16

### 사용 패턴
- 모든 태스크에 긴 메모 작성
- 여정(Journey) 페이지 자주 방문
- 커플 메시지 매일 확인
- 공유 카드 생성 시도

### 피드백
1. **[심각] 사진 업로드 기능이 실제로 작동하지 않음** — "TaskExtendedData에 photos 필드가 있는데 업로드 UI가 없잖아요? 웨딩홀 사진 찍어뒀는데..."
2. **[중간] 여정 타임라인에 날짜 구분이 약함** — "3월 5일에 3개, 3월 7일에 2개 했는데 날짜 경계가 잘 안 보여요."
3. **[중간] 공유 카드가 정적 텍스트만** — "Instagram Story에 올리고 싶은데 디자인이 너무 심플해요. 사진이나 이모지라도..."
4. **[경미] 커플 메시지가 매일 같은 느낌** — "랜덤 메시지인 건 알겠는데, 우리 진행도에 맞는 개인화된 메시지였으면."

## Persona 4: 최민수 (35세, 회계사)
**특성**: 예산 중심. 숫자에 민감. 카테고리별 지출 추적.
**기기**: Desktop (Chrome)

### 사용 패턴
- 데이터베이스에서 비용 열로 정렬
- BudgetChart 자주 확인
- CostBreakdown 퀘스트 클릭으로 필터링
- 모든 태스크에 비용+공급업체 입력

### 피드백
1. **[심각] 예산 총액 설정이 어디있는지 모르겠음** — "BudgetChart에 '총 예산' 원이 있는데, 이걸 바꾸는 UI를 못 찾겠어요. Welcome에서 했나?"
2. **[중간] 비용 입력에 '예상 비용 vs 실제 비용' 구분이 없음** — "typicalCostMin/Max 데이터가 있는데 내 실제 비용과 비교를 보여주면 좋겠어요."
3. **[중간] CostBreakdown 차트가 금액 단위를 '만원'으로 표시하는데 정확한 수치도 보고 싶음** — "차트 hover 시 정확한 원 단위를 보여줘야."
4. **[경미] PriorityProgress가 비용과 연결 안 됨** — "높은 우선순위 미완료 작업의 예상 비용 합이 얼마인지 알고 싶어요."

## Persona 5: 한예린 (26세, 간호사)
**특성**: 바쁜 일정. 짧은 시간에 빠른 확인. 모바일 only.
**기기**: iPhone 14

### 사용 패턴
- 출퇴근 시 "오늘" 탭만 빠르게 확인
- Quick Complete 위주
- 알림/리마인더 원함
- 다른 탭은 거의 안 감

### 피드백
1. **[심각] 오늘 탭이 너무 길어서 스크롤이 많음** — "커플메시지 → 스트릭 → 마일스톤 → 주간진행 → 챌린지 → 추천 → 방치알림 → 할일... 실제 '할 일'을 보려면 한참 내려야 해요."
2. **[심각] 푸시 알림이 없음** — "웹앱이라 알림이 안 오니까 앱을 까먹어요. PWA라도 알림 보내줬으면."
3. **[중간] 헤더 영역이 sticky인데 높이가 큼** — "ProgressRing + 탭바가 상단에 고정되니까 실제 컨텐츠 영역이 좁아요."
4. **[경미] Quick Complete 후 실행취소(undo)가 없음** — "실수로 다른 태스크 완료 눌렀는데 되돌릴 수가 없어요."

## Persona 6: 정태호 (30세, 마케터)
**특성**: 데이터 기반 의사결정. 비교 분석 선호. 대시보드형 사고.
**기기**: Desktop (Firefox)

### 사용 패턴
- Stats 탭 집중
- QuestComparison에서 병목 파악
- CompletionForecast로 일정 관리
- WeeklyTrend로 생산성 추적

### 피드백
1. **[중간] WeeklyTrend가 8주인데 기간 선택이 안 됨** — "최근 4주만 보고 싶을 때도 있고, 전체 기간도 보고 싶을 때가 있어요."
2. **[중간] CompletionForecast가 선형 예측만 함** — "최근 2주의 속도가 빨라졌는데 전체 평균으로 계산하니까 너무 보수적이에요."
3. **[경미] QuestComparison에서 퀘스트 클릭이 안 됨** — "진행률 낮은 퀘스트 클릭하면 바로 그 퀘스트로 가고 싶어요."
4. **[경미] 히트맵에서 특정 날짜 클릭이 안 됨** — "활동 많은 날을 클릭하면 그 날 뭘 했는지 보고 싶어요."

## Persona 7: 윤서현 (28세, 교사)
**특성**: 커플 함께 사용하고 싶음. 공유와 협업 중시.
**기기**: iPad

### 사용 패턴
- 남자친구와 함께 앱 보면서 계획
- 태스크를 "내 것" vs "상대방 것"으로 나누고 싶음
- 캘린더에서 일정 잡기
- 데이터 Export로 백업

### 피드백
1. **[심각] 태스크 담당자 지정이 없음** — "웨딩홀은 내가, 예복은 남친이 하기로 했는데 누가 뭘 하는지 앱에 표시가 안 돼요."
2. **[중간] 두 기기에서 동시 사용 불가** — "localStorage라서 내 폰에서 한 건 남친 폰에서 안 보여요. 동기화가 안 돼요."
3. **[중간] 캘린더에서 직접 일정 추가가 안 됨** — "특정 날짜에 '웨딩홀 방문' 같은 커스텀 일정을 추가하고 싶어요."
4. **[경미] 공유 카드에 두 사람 이름이 나오긴 하지만** — "각자의 기여도(완료 수)도 보여줬으면."

## Persona 8: 김동현 (33세, 자영업자)
**특성**: 시간 부족. 핵심만 빠르게. 불필요한 것에 짜증.
**기기**: Galaxy A54 (중급폰)

### 사용 패턴
- 필요할 때만 접속 (주 1-2회)
- 한 번에 여러 태스크 몰아서 완료
- 불필요한 카드/알림 숨기고 싶음
- 저사양 기기에서 성능 중요

### 피드백
1. **[심각] 137개 태스크 중 필요없는 것을 숨길 수 없음** — "해외 허니문 안 가는데 '허니문' 퀘스트가 계속 보여요. 비활성화하고 싶어요."
2. **[중간] 한 번에 여러 태스크 일괄 완료가 안 됨** — "주말에 '예복 맞추기', '구두 고르기', '셔츠 구매'를 한꺼번에 했는데 하나씩 눌러야 해요."
3. **[중간] 앱 로딩이 느림** — "React Flow 맵이 무거운 건지 퀘스트 탭에서 버벅여요."
4. **[경미] 오늘 탭의 카드들을 접거나 순서 변경하고 싶음** — "스트릭이나 챌린지는 관심 없는데 항상 위에 있어요."

## Persona 9: 박서아 (31세, 프리랜서)
**특성**: 비선형적 사용. 여러 퀘스트 동시 진행. 체계적이지 않음.
**기기**: MacBook Air

### 사용 패턴
- 퀘스트 맵에서 잠금 해제된 것 중 흥미로운 것부터
- 의존성 무시하고 싶은 욕구
- 메모를 일기처럼 길게 작성
- 여정 페이지에서 자기 기록 리뷰

### 피드백
1. **[중간] 퀘스트 잠금이 답답함** — "'예물/예단'을 하려는데 '상견례'를 먼저 해야 한다고? 우린 이미 했는데 앱에 기록 안 했을 뿐이에요."
2. **[중간] 메모 텍스트 영역이 작음** — "길게 쓰면 스크롤이 생기는데 전체 내용을 보기 어려워요."
3. **[경미] 의존성 그래프가 왜 이렇게 연결됐는지 설명 없음** — "왜 '인테리어'가 '신혼집' 뒤에 와야 하는지 이유를 보여줬으면."
4. **[경미] 여정 타임라인에서 내 메모만 모아보는 필터가 없음** — "사진은 없지만 메모를 적은 것들만 보고 싶어요."

## Persona 10: 이하은 (24세, 대학원생)
**특성**: 아직 결혼 안 했지만 미리 계획. 호기심으로 시작. 저예산.
**기기**: iPhone SE (작은 화면)

### 사용 패턴
- Welcome에서 날짜 미설정 ("아직 안 정해졌어요")
- 여러 퀘스트 구경하며 "이런 것도 해야 하나?" 탐색
- 비용 정보에 관심 (typicalCost)
- 실제 완료보단 학습 목적

### 피드백
1. **[심각] 결혼일 미설정 시 캘린더/D-Day 기능이 거의 비활성** — "날짜를 안 넣으면 캘린더가 텅 비어요. '대략 6개월 후' 같은 상대적 설정이라도..."
2. **[중간] typicalCost(예상 비용) 정보가 태스크 상세에서만 보임** — "각 태스크에 '평균 50~100만원'이라고 있는데, 전체 합산이 얼마인지 한눈에 보고 싶어요."
3. **[중간] iPhone SE에서 캘린더 셀이 너무 좁음** — "날짜 하나에 텍스트가 전혀 안 보이고 숫자 배지만 보여요."
4. **[경미] "이 앱 뭐하는 앱이야?" 온보딩 설명이 부족** — "Welcome이 바로 이름 입력부터 시작하는데, 이 앱이 뭘 해주는지 먼저 보여줬으면."
