# Cycle 21 — Clarify Answers

1. **핵심 가치**: "나 이만큼 했다"를 보여주는 성취 카드. 커플 이름 + 진행률 + D-Day.

2. **공유 형태**: Canvas API로 이미지 생성 → Blob → Web Share API (모바일) or 다운로드 (데스크톱). html2canvas는 의존성 추가 대비 이득이 적음 — 직접 Canvas 드로잉.

3. **이미지 콘텐츠**:
   - 커플 이름 ("현종 ♥ 영희")
   - D-Day 카운트 ("D-120")
   - 전체 진행률 (원형 진행 바 + 퍼센트)
   - 완료 퀘스트 수 / 전체 퀘스트 수
   - MarryRoad 워터마크

4. **Canvas API 충분**: 텍스트 + 도형 + 그라데이션. 복잡한 HTML 레이아웃 불필요.

5. **바이럴 문구**: "우리의 결혼 준비 XX% 완료!" — 감성적이면서 구체적.
