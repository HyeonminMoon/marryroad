# Cycle 1 - Idea Candidates

## Scoring: Impact(1-10) / Novelty(1-10) / Effort(1-10, lower=easier)

### 1. Wedding Journey Timeline
**Description**: A visual timeline page that turns completed tasks into a narrative story with dates, photos, memos, and milestone markers.
- Impact: **10** - Creates the emotional "look back" moment
- Novelty: **9** - No wedding app does this with gamified data
- Effort: **4** - Uses existing data, new UI only
- **Score: 25 (10+9+(10-4)+1)**

### 2. D-Day Countdown Dashboard
**Description**: Smart countdown that shows what should be done by each milestone date, with urgency signals.
- Impact: **7** - Practical but not emotional
- Novelty: **5** - Many apps have countdowns
- Effort: **3** - Simple calculation + UI
- **Score: 19**

### 3. Partner Sync (2-player mode)
**Description**: Share progress between partners, assign tasks, see each other's contributions.
- Impact: **8** - Huge value for couples
- Novelty: **7** - Rare in wedding apps
- Effort: **9** - Requires auth + realtime + backend
- **Score: 16**

### 4. Vendor Book
**Description**: Aggregated vendor contacts, ratings, and costs across all tasks in one view.
- Impact: **6** - Useful but utilitarian
- Novelty: **4** - Contact lists exist
- Effort: **3** - Data aggregation + new page
- **Score: 17**

### 5. Budget Trend Visualization
**Description**: Graph showing spending over time, projected vs actual, category breakdown.
- Impact: **7** - Financial clarity reduces anxiety
- Novelty: **5** - Budget trackers exist
- Effort: **4** - Chart library + data aggregation
- **Score: 18**

### 6. Achievement System
**Description**: Unlock badges for milestones ("First Quest Complete", "Budget Under Control", "50% Done").
- Impact: **6** - Fun but surface-level
- Novelty: **6** - Common in gamification
- Effort: **3** - Static data + UI
- **Score: 19**

### 7. Emotional Letter Generator
**Description**: AI generates a personalized letter/vow draft based on relationship milestones and memos entered.
- Impact: **9** - Deeply emotional
- Novelty: **10** - Unique concept
- Effort: **7** - Requires AI API integration
- **Score: 22**
